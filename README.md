🤖 6-DOF Robotic Arm Control System
This project features an end-to-end automation solution for teaching, recording, and executing precise sequences on a 6-Degree-of-Freedom robotic arm powered by STS3215 smart serial servos. The core philosophy of this project is to eliminate complex terminal interfaces and bridge low-level hardware control with a clean, responsive, and intuitive web-based SCADA dashboard built on Node-RED, running entirely on a Raspberry Pi.

The system features an automated failover driver mechanism that intelligently scans, detects, and reconnects to active USB ports like /dev/ttyACM0 or /dev/ttyACM1, ensuring zero communication downtime. It implements a highly efficient multi-axis movement architecture using the driver's native synchronous protocol. Instead of updating motors sequentially, the system packs target positions, acceleration, and speed into unified data frames. It dynamically calculates the maximum joint distance to adjust real-time delay timings, guaranteeing perfectly synchronized, smooth, and jerk-free movements.

The user interface acts as a custom SCADA system with intuitive touch controls. The Torque Off feature releases motor tension, allowing hand-guided positioning for kinesthetic teaching. A specialized JavaScript pipeline filters raw data streams, hiding complex joint coordinates and displaying simple, readable logs on the panel. Precision sliders allow live adjustment of execution speed and acceleration. When the operator triggers the execution, Node-RED packs the configured speed, acceleration, and the entire recorded multi-coordinate sequence into a json file. The execution script then reads this file asynchronously, locking the torque registers and driving the arm flawlessly through the recorded path.

To run the project, clone the repository to your Raspberry Pi, set up a Python virtual environment, and install the required dependencies. After that, open your local Node-RED editor, use the Import feature to paste the contents of the node_red_flow.json file, and click Deploy. Connect your serial bus smart servos through your hardware interface adapter into any available USB port on the Raspberry Pi, and the driver will handle the port allocation automatically.
